# Skills Without Borders (SWB)

A Streamlit + Supabase prototype to document and verify skills, match opportunities, and demonstrate a deployable stack (Streamlit / Replit / Supabase / Docker).

See docs in the repository for setup, Supabase SQL schema, and deployment options (Streamlit Cloud, Docker, Replit).

Quick start:

1. Copy `.env.example` to `.env` and fill SUPABASE_URL and SUPABASE_KEY.
2. Create a Python virtualenv and install requirements: `pip install -r requirements.txt`.
3. Run locally: `streamlit run app.py`.
